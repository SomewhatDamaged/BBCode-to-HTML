# BBCode-to-HTML
A simple Flet app that converts a BBCode document to HTML.

Tags supported:
- `[b]` (bold) -> `<strong>`
- `[i]` (italic) -> `<em>`
- `[quote]` -> `<blockquote>`
- `[hr]` -> `<hr />`

Detects paragraphs by double-linefeed (double-spaced paragraphs), single linefeeds are replaced with `<br />`
